# AI-powered-resume-intelligence-platform

AI-powered resume intelligence platform for ATS scoring, resume analysis, job matching, and interview readiness.

## Features
- ATS score analysis
- Resume optimization suggestions
- Job matching and comparison
- Cover letter generation
- Interview question practice

## Getting started
1. Install dependencies: npm install
2. Create a local environment file: .env.local
3. Start the app: npm run dev

Open http://localhost:3000 to view it locally.
